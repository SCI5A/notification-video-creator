# 📱 مُنشئ فيديو الإشعارات | Notification Video Creator Template

قالب موقع ويب متكامل ومتجاوب (Responsive) يتيح للمستخدمين صناعة فيديوهات إشعارات احترافية للهواتف مباشرة من المتصفح.

## 🚀 المعاينة الحية (Live Demo)
[🔗 اضغط هنا لمشاهدة التطبيق يعمل مباشرة](https://SCI5A.github.io/notification-video-creator/)

## ✨ الميزات
- 🛠️ **بدون خادم (Serverless):** يعتمد بالكامل على المتصفح عبر Canvas و MediaRecorder API.
- 🎨 **تخصيص كامل:** تغيير الخلفيات، الخطوط (Cairo/Tajawal)، وإضافة صور مخصصة.
- 🎬 **تسجيل فيديو عالي الجودة:** لايدعم التصديربدقات متعددة (720p, 1080p).
- 🌐 **دعم كامل للعربية:** واجهة موجهة بالكامل من اليمين لليسار (RTL).

## 💻 طريقة التشغيل محلياً
1. قم بعمل `Clone` للمستودع.
2. افتح ملف `index.html` في أي متصفح حديث، أو استخدم إضافة `Live Server` في VS Code.

## ⚙️ التعديل والتطوير
- لتعديل واجهة المستخدم: عدل ملف `index.html` (يحتوي على الأنماط المضمنة).
- لتعديل تأثيرات الرسوم المتحركة أو أبعاد الفيديو: عدل الأكواد البرمجية داخل `index.html`.
